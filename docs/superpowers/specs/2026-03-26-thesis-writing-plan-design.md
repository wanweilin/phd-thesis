# 博士论文写作计划设计

## 概述

将三篇已发表论文（NAP, CADS, SWaST）整合为一篇以"深度神经网络全流程数据选择"为主线的博士毕业论文。

- **写作语言**：中文（术语首次出现附英文原文）
- **截止日期**：2026年9月底
- **工作方式**：在当前 repo 中直接写内容，格式/模板后续整理
- **LaTeX模板**：已有学校模板，内容完成后再套用

---

## 核心策略：自顶向下四层细化

```
第1轮：全文骨架（6章标题 + 各章1句话定位）
第2轮：章节骨架（每章的小节划分 + 每小节1句话摘要）
第3轮：段落骨架（每小节的段落要点 / bullet points）
第4轮：正文填充（将要点扩展为正式中文段落）
```

每一轮都是一次完整的全文遍历，层层递进。每轮完成后做一次回顾，确保叙事一致性。

## 各阶段详细说明

### Phase 0: 准备（1周）

- [ ] 通读三篇论文，列出符号差异表
- [ ] 确定统一符号方案（以哪篇为基准，其余适配），输出 `thesis/notation.md`
- [ ] 在 repo 中创建 `thesis/` 目录，按章节建立文件结构
- [ ] 创建 `PROGRESS.md` 进度追踪文件

### Phase 1: 全文骨架（1周）

- [ ] 确认 THESIS_PURPOSE.md 中的6章大纲
- [ ] 为每章写1句话定位（在数据选择主线下的角色）
- [ ] 设计章节间的过渡逻辑（每个过渡点1-2句话描述）
- [ ] 绘制"研究地图"概念图（三篇工作在深度学习流程中的位置）

### Phase 2: 章节骨架（2周）

逐章确定小节划分：

**第一章 绪论**
- 1.1 研究背景
- 1.2 研究问题
- 1.3 本文贡献概览
- 1.4 论文结构

**第二章 相关工作**
- 2.1 训练数据选择与coreset方法
- 2.2 模型剪枝与计算效率
- 2.3 双层优化在机器学习中的应用
- 2.4 分布外检测
- 2.5 数据选择在推理阶段的延伸

**第三章 CADS**
- 依据 arXiv-2510.16806v2 结构，删去 related work，补充章节引言和小结

**第四章 SWaST**
- 依据 arXiv-2511.09901v2 结构，删去 related work，补充章节引言和小结

**第五章 NAP**
- 依据 arXiv-2402.18162v4 结构，删去 related work，补充章节引言和小结

**第六章 总结与展望**
- 6.1 工作总结
- 6.2 统一视角
- 6.3 局限性
- 6.4 未来方向

每小节附1-2句话内容摘要。

### Phase 3: 段落骨架（3周）

- 逐小节列出段落要点（bullet points）
- 每个 bullet point 标注来源：
  - `[新写]` — 需要全新撰写
  - `[改编自论文X §Y]` — 从小论文改编
  - `[翻译自论文X §Y]` — 从小论文翻译
- 章节引言和过渡段落标注为 `[新写]`
- 核心方法/实验部分标注为 `[改编]` 或 `[翻译]`

### Phase 4: 正文填充（6周）

- 推荐章节顺序：第三章 → 第四章 → 第五章（改编章节先行，积累手感）→ 第一章 → 第二章 → 第六章（原创章节在对全文理解更深后撰写）
- 每次聚焦一个小节，提供对应小论文原文作为参考
- 填充时遵循 WRITING_GUIDELINES.md 中的写作规范
- 涉及实验数字的段落标注 `[VERIFY]`，在 Phase 5 统一核实
- 每章完成后做一次叙事一致性检查

### Phase 5: 整合润色（3周）

- 统一全文符号和术语
- 检查各章过渡是否自然
- 统一图表格式和编号
- 合并三篇论文的 .bib 文件，去重
- 核实所有 `[VERIFY]` 标记的实验数字
- 查重自检（即使引用自己的论文也需改写以降低重复率）
- 全文通读润色

### Phase 6: 审阅修改（~10周）

- **第1-2周**：初稿提交导师审阅
- **第3-4周**：收到反馈，第一轮修改
- **第5-6周**：第二轮导师反馈
- **第7-8周**：最终修改 + 套用学校 LaTeX 模板
- **第9-10周**：格式检查 + 查重 + 提交盲审

补充内容（可在 Phase 5 或 Phase 6 期间完成）：
- [ ] 中英文摘要
- [ ] 致谢
- [ ] 附录（较长的实验细节、额外消融实验等）
