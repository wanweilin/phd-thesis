# 第一章 绪论 — 段落要点

---

## 1.1 研究背景：深度学习对数据的依赖与数据选择的核心地位

**段落1 — 数据驱动范式的成功与代价**
- 以 ImageNet 时刻为起点，深度学习在视觉、语言、多模态等领域的突破均建立在大规模数据之上（举 2-3 个里程碑：AlexNet、GPT 系列、扩散模型等）。
- 在大模型时代，数据的地位进一步凸显：LLM 的许多能力涌现被归因于训练数据的规模与质量（而非单纯的模型架构改进）；高质量数据的获取与筛选已成为竞争核心。Ilya Sutskever 曾提出"数据即将耗尽"的警示——无论这一判断是否准确，它反映了前沿研究对数据资源的高度依赖与焦虑，也暗示了"更聪明地使用数据"（而非仅仅"更多数据"）的迫��性。
- 但数据规模增长带来现实瓶颈：计算成本随数据量超线性增长，高质量标注日益稀缺且昂贵，大量收集的数据中存在冗余、噪声和分布不均。
- 落脚点：并非所有数据都同等有价值——如何识别和利用最相关的数据，成为提升深度学习效率与可靠性的关键问题。

**段落2 — 数据选择的提出**
- 定义数据选择（Data Selection）：在深度学习的不同阶段，根据特定目标从候选数据中识别、筛选或评估最相关样本的过程。
- 数据选择的广义性：不仅指训练阶段的子集选择（coreset selection），还包括数据混合配比（data mixture）、数据质量过滤，以及推理阶段对输入数据有效性的判断。
- 区分于数据增强和数据集蒸馏：数据选择是从现有数据中"选"，而非"造"新数据。

**段落3 — 数据选择的多阶段性（引出主线 A）**
- 训练阶段：从全量数据中选出最具信息量的子集，在有限计算资源下最大化模型性能。
- 微调阶段：面对多来源、多质量的候选数据，选择最匹配下游任务的数据子集或配比，在有限微调预算下高效适配。
- 推理阶段：模型部署后，判断输入样本是否处于模型的"有效数据域"内，即推理时的数据相关性判断。
- 小结：数据选择贯穿深度学习的全流程，但在不同阶段以不同形式出现，面临不同约束。

**段落4 — 数据选择的交互性（引出主线 B）**
- 核心论点：数据选择的最优策略并非孤立决定的，而是取决于与之交互的关键因素。
- 三个关键交互维度的预告（不展开，仅点明）：计算资源约束影响最优数据子集的组成，模型容量约束与数据冗余存在耦合，推理时分布偏移要求对数据有效性做在线判断。
- 现有方法的共性不足：大多数数据选择研究将问题孤立处理，忽视这些交互因素的影响。本文旨在填补这一空白。

## 1.2 研究问题

**段落1 — 总述：双视角下的三个子问题**
- 承接 1.1 的两个视角，明确本文围绕数据选择提出三个子问题。
- [表1-1: 三个研究问题的双视角定位（研究问题 × 主线A阶段视角 × 主线B交互变量视角）]
- 说明三个子问题的递进关系：从训练侧到推理侧，交互维度从计算资源到模型容量再到分布偏移。

### 1.2.1 子问题一：计算预算约束下的训练数据选择

- 问题陈述：给定有限的计算预算，如何从全量训练数据中选出使模型性能最优的子集？
- 阶段定位：这是训练/微调阶段最基础的数据选择问题。
- 交互维度：已有研究表明，不同计算预算下最优数据子集的组成截然不同——低预算倾向简单样本，高预算可从多样化数据获益。现有方法忽视这一交互，导致没有算法能跨预算一致领先。
- 待解决的核心挑战：如何将计算预算作为数据选择的显式变量，而非隐含假设。

### 1.2.2 子问题二：数据选择与模型剪枝的协同

- 问题陈述：当训练数据选择与模型剪枝同步进行时，如何利用二者的交互实现协同优化？
- 阶段定位：同属训练阶段，但将"选择"的维度从数据扩展到模型。
- 交互维度：数据冗余与参数冗余存在双向耦合——冗余样本导致模型为拟合噪声而保留不必要权重（阻碍剪枝），冗余参数扩大了候选空间导致 coreset 选择失效（干扰选择）。
- 待解决的核心挑战：现有工作将数据选择和模型剪枝视为独立流水线，未利用也未分析二者的耦合关系。

### 1.2.3 子问题三：推理时的数据有效性判断

- 问题陈述：模型部署后，如何判断输入样本是否处于训练数据选择所界定的"有效数据域"内？
- 阶段定位：推理阶段的数据选择——OOD 检测本质上是在推理时对输入数据做"选择"。
- 交互维度：训练阶段的数据选择界定了模型的有效域边界。当推理时输入分布发生偏移，需要判断输入是否仍属于该有效域——这是数据选择与推理时分布偏移的交互。
- 待解决的核心挑战：现有后处理方法利用全局池化后的特征，丢失了池化前通道内的分布信息，限制了检测能力。

## 1.3 本文贡献概览

**段落1 — 三项工作的统一定位**
- 本文围绕数据选择这一核心方法论，针对上述三个子问题分别提出一个框架/方法，共同构成对数据选择问题从训练到推理的系统性研究。

**贡献一：CADS — 计算预算感知的数据选择（第三章）**
- 提出 CADS（Computation-Aware Data Selection）框架，将计算预算作为双层优化中的显式约束。
- 技术贡献：无 Hessian 策略梯度 + 惩罚松弛将双层优化转化为高效单层目标，实现 3-20× 加速。[VERIFY]
- 两种变体：CADS-E（样本级）和 CADS-S（来源级），在视觉和语言任务上最高提升 14.42%。[VERIFY]
- 发表于 NeurIPS 2025。

**贡献二：SWaST — 数据与模型的协同优化（第四章）**
- 提出 SWaST（Simultaneous Weight and Sample Tailoring）框架，通过交替执行权重剪枝与 coreset 选择建立协同效应。
- 技术贡献：首次分析数据冗余与参数冗余的双向耦合，发现并解决"双重损失"问题，引入状态保护机制。
- 最高提升 17.83%，同时实现 10%-90% FLOPs 削减。[VERIFY]
- 发表于 AAAI 2026。

**贡献三：NAP — 基于神经激活先验的 OOD 检测（第五章）**
- 提出 NAP（Neural Activation Prior），观察到 ID 样本在全连接层前的通道内激活少量神经元的概率显著高于 OOD 样本。
- 技术贡献：基于该先验设计即插即用的评分函数，与现有方法正交互补，无需额外训练。
- 在 CIFAR 和 ImageNet 基准上将 FPR95 最高降低 66.03%。[VERIFY]
- 投稿中。
