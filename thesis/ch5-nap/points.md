# 第五章 推理阶段的数据相关性判断 — 段落要点

---

## 5.1 章节引言：推理时分布偏移下的数据相关性判断

**段落1 — 从训练侧到推���侧的���换**
- 前两章解决的核心问题：在训练阶段，如何选择最优数据子集（第三章）以及如何与模型压缩协同（第四章）。训练阶段的数据选择从根本上界定了模型的能力边界——模型只在训练数据所覆盖的分布上可靠。
- 模型部署后，输入数据的来源不再可控。当输入样本偏离训练分布时，模型可能产生过度自信的错误预测。因此需要一个"守门人"机制，判断输入是否处于模型的有效数据域内。

**段落2 — OOD 检测作为推理时的数据选择**
- 分布外检测（OOD Detection）可以被理解为推理时的数据"选择"：选择（接受）属于训练分布内的输入，拒绝偏离训练分布的输入。
- 与训练时数据选择的呼应：训练选择界定有效域的边界，推理检测守护这一边界。从"数据选择"的统一视角看，二者是同一问题在不同阶段的体现。

**段落3 — 现有方法的局限与本章目标**
- 现有后处理 OOD 检测方法利用的信号主要来自：最终输出层（MSP、Energy）或全局池化后的中间特征（Mahalanobis、KNN、ReAct、DICE、ASH）。
- 共同的信息盲区：全局平均池化将 $C \times H \times W$ 压缩为 $C$ 维，丢失了通道内的空间激活分布信息。
- 本章提出神经激活先验（Neural Activation Prior, NAP）：利用这一被忽视的池化前通道内信息，提供与现有方法正交的新检测信号。

## 5.2 基础知识：OOD 检测的典型设定

**段落1 — 问题定义**
- 形式化：给���在 ID 数据集 $\mathcal{D}_{\text{in}}$ 上训练的分类器 $f$，目标是设计评分函数 $S(x; f)$，使得 $S(x_{\text{in}}) > S(x_{\text{out}})$（ID 样本得分更高）。通过阈值 $\tau$ 实现判别：$S(x) \geq \tau$ 则接受，否则拒绝。
- 后处理方法的约束：不修改模型训练过程，不需要 OOD 数据参与训练，仅利用已训练模型的推理时输出。这一约束具有实际意义——部署后的模型通常不允许重新训练。

**段落2 — 评价指标**
- FPR95（False Positive Rate at 95% True Positive Rate）：在保证 95% 的 ID 样本被正确接受的前提下，误接受 OOD 样本的比例。越低越好。
- AUROC（Area Under the Receiver Operating Characteristic curve）：综合衡量在不同阈值下的区分能力。越高越好。
- 为何 FPR95 更受关注：在安全关键应用中，高 TPR 是刚需（不能拒绝太多正常输入），因此在固定高 TPR 下的 FPR 更具实际意义。

## 5.3 神经激活先验（Neural Activation Prior）

**段落1 — 核心观察**
- 观察对象：全连接层前的特征图，形状为 $C \times H \times W$（$C$ 个通道，每个通道是 $H \times W$ 的空间特征图）。
- 核心发现：对于 ID 样本，每个通道内少量空间位置具有显著高于均值的激活值（尖锐激活模式）；对于 OOD 样本，通道内的激活更均匀分散，缺乏尖锐响应。
- 量化：定义两个逐通道统计量——$\text{Max}(A_j) = \max_{k,l} A_{jkl}$（通道内最大激活）和 $\text{Mean}(A_j) = \frac{1}{HW}\sum_{k,l} A_{jkl}$（通道内均值激活）。ID 样本的 Max/Mean 比值显著高于 OOD 样本。

**段落2 — 直觉解释**
- 为何 ID 样本产生尖锐激活：训练过程使得每个通道的滤波器学会检测特定的视觉模式（边缘、纹理、物体部件等）。ID 样本包含这些模式，触发对应滤波器在特定空间位置产生强响应。
- 为何 OOD 样本激活更分散：OOD 样本不包含训练数据中的典型模式，滤波器无法找到精准匹配，激活值在空间上更均匀且整体较弱。
- NAP 是训练的"自然副产品"：不需要任何额外训练或 OOD 数据，这一先验在标准训练的模型中自然存在。

**段落3 — 为何选择倒数第二层**
- 浅层特征（conv1、早期模块）捕获低级特征（边缘、颜色），ID 和 OOD 共享这些基础特征，区分度低。
- 深层特征（全连接层前）捕获高级语义特征，对训练数据分布高度特异，ID/OOD 差异最大。
- 实证验证：NAP 在不同层上的区分度随深度递增。[VERIFY: 层级分析实验]

**段落4 — 向 Transformer 的扩展**
- Vision Transformer 中无传统全局池化层，但 NAP 的思想可迁移：cls token 对其他 token 的注意力权重反映了模型对不同输入区域的"关注集中度"。
- ID 样本的注意力更集中（少数 token 获得高权重），OOD 样本的注意力更分散。
- 由于注意力权重归一化后均值为常数 $1/(l+1)$，NAP 在 Transformer 中简化为关注最大注意力权重 $\text{Max}(A)$。

## 5.4 方法：基于 NAP 的评分函数设计

### 5.4.1 CNN 评分函数

**段落1 — 评分函数定义**
- 受信噪比（SNR）启发的评分：$S_{\text{NAP}}(x; f) = \frac{1}{C}\sum_{j=1}^{C}\left(\frac{\text{Max}(A_j)}{\text{Mean}(A_j) + \epsilon}\right)^2$。
- Max 作为"信号"（目标模式的响应强度），Mean 作为"噪声"（背景激活水平），平方放大差异。$\epsilon = 1.0$ 保证数值稳定。

**段落2 — 设计动机**
- 为何不直接用 Max：不同通道的激活尺度差异很大，直接取 Max 会被高激活尺度通道主导。除以 Mean 实现逐通道归一化，使所有通道的贡献更均衡。
- 为何取平方：Max/Mean 比值在 ID 和 OOD 之间的差异通过平方被非线性放大，提高区分度。
- 跨通道平均：聚合所有通道的信息，减少单个通道的随机波动。

### 5.4.2 Transformer 扩展

**段落1 — 评分函数**
- $S_{\text{NAP}}^{\text{Former}} = \text{Max}(A)$：取最终 Transformer 块中 cls token 注意力权重的最大值。

**段落2 — 简化的原因**
- 注意力权重经 softmax 归一化后，均值为固定常数 $1/(l+1)$（$l$ 为 token 数量）。因此 SNR 形式 Max/Mean 退化为 Max 的常数倍，Mean 的归一化作用已被 softmax 隐式完成。

### 5.4.3 与现有方法的组合

**段落1 — 加权几何均值组合**
- 组合形式：$S_{\text{NAP-}\star}(x; f, w) = -S_{\star}(x; f)^w \cdot S_{\text{NAP}}(x; f)^{1-w}$，其中 $\star$ 代表 MSP、Energy、ReAct 等任意现有方法，$w \in [0,1]$ 为权重。
- 几何均值的选择：相比线性组合，几何均值对尺度差异更鲁棒（两个评分的量纲不需要一致）。

**段落2 — 正交互补性**
- 信息来源的正交：NAP 利用池化前的通道内分布信息（空间维度），现有方法利用池化后的全局特征或输出层信息（通道维度）。二者捕获不同层面的 ID/OOD 差异。
- 实证：NAP 与几乎所有测试的基线方法组合后都能提升性能，说明 NAP 确实提供了互补的检测信号。[VERIFY: 组合提升的具体数字]
- 权重参数 $w$ 的优化：通过数据增强构造伪 OOD 样本，用二分搜索确定最优 $w$，无需真实 OOD 数据。

## 5.5 实验

**段落1 — 实验设置**
- ID 数据集：CIFAR-10、CIFAR-100、ImageNet-1K。
- OOD 数据集：SVHN、Textures、iSUN、LSUN（Crop/Resize）、Places365（CIFAR 基准）；iNaturalist、SUN、Places、Textures（ImageNet 基准）。
- 网络架构：DenseNet、ResNet-50、MobileNetV2、VGG-16（CNN）；ViT-B/16（Transformer）。
- 基线方法：MSP、Energy、ODIN、ReAct、DICE、ASH、KNN、Mahalanobis。
- NAP 组合变体：NAP-M、NAP-E、NAP-R、NAP-D、NAP-A、NAP-K。[VERIFY: 完整设置]

**段落2 — CIFAR 基准结果**
- CIFAR-10：NAP-K 将 FPR95 从 15.05% 降至 7.79%；NAP-E 实现最大相对降幅 66.03%。[VERIFY: 详细数据]
- CIFAR-100：NAP-R 将 FPR95 从 41.40% 降至 25.71%，最大相对降幅 58.71%。[VERIFY]
- 关键观察：NAP 的提升在不同 OOD 数据集上不均匀，对 Textures 等纹理类 OOD 改善最大（这类 OOD 在池化后特征上与 ID 难以区分，但在通道内激活模式上差异显著）。

**段落3 — ImageNet 基准结果**
- 大规模验证：ImageNet-1K 作为 ID，四个 OOD 数据集。
- NAP-A（与 ASH 组合）将平均 FPR95 从 35.66% 降至 29.86%。[VERIFY: 详细数据]
- 在 Textures 和 iNaturalist 上改善最为显著，与 CIFAR 上的观察一致。

**段落4 — Transformer 实验**
- ViT-B/16 上的结果：NAP-M 将 FPR95 从 61.72%（MSP）降至 54.40%。[VERIFY]
- 验证了 NAP 思想（激活集中度作为 ID/OOD 信号）从 CNN 到 Transformer 的迁移性。

**段落5 — 层级分析**
- 在 DenseNet 的不同层（conv1、transition1-3、penultimate）分别计算 NAP 评分。
- 结果：ID/OOD 区分度随网络深度单调递增，倒数第二层效果最好。浅层特征对 ID/OOD 无区分力（共享低级特征）。
- 验证了 NAP 的理论直觉：深层特征更具 ID 数据特异性。[VERIFY: 层级分析图]

## 5.6 章节小结

**段落1 — 本章贡献总结**
- 发现并形式化了神经激活先验（NAP）：池化前通道内的激活集中度是区分 ID 和 OOD 的有效信号，这一信号被现有方法忽视。
- 基于 NAP 设计了即插即用的评分函数，适用于 CNN 和 Transformer，与现有方法正交互补。
- 在 CIFAR 和 ImageNet 基准上，NAP 与多种基线组合后均实现显著提升，FPR95 最高降低 66.03%。[VERIFY]
