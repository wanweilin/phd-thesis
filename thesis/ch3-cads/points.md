# 第三章 计算预算感知的训练数据选择 — 段落要点

---

## 3.1 章节引言：预算约束下数据选择的必要性

**段落1 — 从第二章过渡：现有方法的盲区**
- 第二章指出现有训练数据选择方法隐含假设最优子集独立于计算预算。本章从一个直觉出发挑战这一假设：在只能训练 10 轮和可以训练 100 轮时，"最有价值的数据"应当是相同的吗？
- 事实上，低预算下模型仅能学习简单模式，复杂/噪声样本反而有害；高预算下模型有能力从多样化数据中提取更丰富信息。这意味着最优数据子集依赖于可用预算。

**段落2 — 探索性实验：预算改变最优数据**
- 设计探索性实验：在异构多源数据上，固定数据选择策略（如始终选干净源 / 始终均匀混合），然后在不同计算预算下评估。[VERIFY: 实验设置——CIFAR-10, 5 个噪声水平的数据源]
- [图3-1: 不同计算预算下低频与高频数据训练的验证损失对比（对应原论文 Fig.1）。需手绘。]
- 关键结果：没有任何固定策略能跨预算一致领先。低预算下偏好干净数据的策略最优，高预算下纳入更多源的策略更优。[VERIFY: 具体数字与图]
- 结论：最优数据选择策略是计算预算的函数，预算应作为数据选择的显式输入。

**段落3 — 本章目标**
- 提出 CADS（Computation-Aware Data Selection）框架，核心思想：将计算预算 $C$ 作为双层优化中的显式约束，让数据选择策略自动适应给定预算。
- 预告两个关键技术挑战：(1) 预算约束导致内层非收敛，标准双层优化方法失效；(2) 需要高效求解以避免选择过程本身消耗过多预算。

## 3.2 问题定义：计算约束下的数据选择建模

**段落1 — 形式化定义**
- 符号定义：训练集 $\mathcal{D}_{\text{trn}}$、验证集 $\mathcal{D}_{\text{val}}$、计算预算 $C$（以前向-反向传播次数度量）、数据选择掩码 $\mathbf{m} \in \{0,1\}^N$。
- 双层优化形式：外层 $\min_{\mathbf{s}} \mathbb{E}_{\mathbf{m} \sim p(\mathbf{m}|\mathbf{s})}[\mathcal{L}_{\text{val}}(\theta_C(\mathbf{m}))]$，其中 $\theta_C(\mathbf{m})$ 表示在子集 $\mathbf{m}$ 上训练恰好 $C$ 步后的模型参数。
- 与标准数据选择问题的关系：当 $C \to \infty$ 时退化为标准 coreset 选择问题。

**段落2 — 预算约束带来的根本困难**
- 关键区别：标准双层优化假设内层求解到最优 $\theta^*(\mathbf{m})$，可利用隐式函数定理计算外层梯度。但预算约束下内层仅运行 $C$ 步，$\theta_C(\mathbf{m}) \neq \theta^*(\mathbf{m})$，隐式微分的前提不成立。
- 替代方案（迭代微分/截断反向传播）要求对 $C$ 步训练做反向传播，内存和计算开销与 $C$ 成正比，对大预算不可行。
- 因此需要一种不依赖内层最优性、也不需要对训练过程做梯度回传的新方法。

## 3.3 方法：CADS 框架

### 3.3.1 双层优化建模与策略梯度基线（Bilevel-CADS）

**段落1 — 概率化重参数化**
- 将离散掩码 $\mathbf{m}$ 的优化转化为连续参数 $\mathbf{s}$ 的优化：定义参数化采样分布 $p(\mathbf{m}|\mathbf{s})$，外层目标变为关于 $\mathbf{s}$ 的期望优化。
- 这一重参数化将组合优化问题（$2^N$ 个候选子集）转化为连续优化问题。

**段落2 — 策略梯度估计**
- 利用 REINFORCE 策略梯度：$\nabla_{\mathbf{s}}\mathbb{E}[\mathcal{L}_{\text{val}}] = \mathbb{E}[\mathcal{L}_{\text{val}} \cdot \nabla_{\mathbf{s}} \log p(\mathbf{m}|\mathbf{s})]$。
- 核心优势：无需对训练过程做梯度回传（无 Hessian），外层梯度仅需评估验证损失和采样分布的对数梯度。
- 与强化学习的类比：数据选择 $\mathbf{m}$ 是"动作"，验证损失是"奖励"，$\mathbf{s}$ 是"策略参数"。

**段落3 — Bilevel-CADS 的局限**
- 每次外层更新需要 $K$ 次独立的完整训练（采样 $K$ 个不同的 $\mathbf{m}$ 并各训练 $C$ 步），总计算量为 $K \times C$。
- 即使 $K=5$，在中等预算下也需 5 倍完整训练的开销，实际中不可接受。
- Bilevel-CADS 作为概念验证（proof of concept），验证了预算感知数据选择的有效性，但需要更高效的实现。

### 3.3.2 惩罚松弛与单层目标

**段落1 — 核心思想：将双层转化为单层**
- 引入惩罚项：$\mathcal{L}^{\alpha}_{\text{CADS}} = \mathbb{E}_{p(\mathbf{m}|\mathbf{s})}[\mathcal{L}_{\text{val}}(\theta) + \alpha(\mathcal{L}_{\text{trn}}(\theta, \mathbf{m}) - l(|\mathbf{m}|))^2]$。
- 直觉：惩罚项要求模型在当前子集上的训练损失接近该子集大小对应的"目标损失" $l(|\mathbf{m}|)$——即模型应充分利用所选数据（不欠拟合也不过拟合）。
- 将双层结构消解为单层联合优化 $(\theta, \mathbf{s})$，可用标准 SGD 交替更新。

**段落2 — 一维损失近似 $l(|\mathbf{m}|)$**
- $l(|\mathbf{m}|)$ 是"在大小为 $|\mathbf{m}|$ 的子集上训练 $C$ 步后的最小可达训练损失"。直接求解需要对每个子集大小做完整训练，不可行。
- 关键发现：$l(|\mathbf{m}|)$ 关于子集大小近似呈对数线性关系（log-linear）。只需在 3-5 个不同子集大小上做短训练，即可拟合该函数。[VERIFY: 拟合细节和精度]
- [图3-3: 一维损失近似的拟合效果——不同子集大小下的实际训练损失与拟合曲线对比（对应原论文 Fig.6/附录）。需手绘。]
- 这一近似将惩罚项的计算从"每个候选子集独立训练"降低为"一次性拟合 + 查表"，是 CADS 效率提升的关键。

**段落3 — 效率分析**
- CADS 的总额外开销：$(5/A + 2\gamma)C$，其中 $A$ 为摊销因子，$\gamma < 1$ 为验证比例。[VERIFY: 具体公式]
- 与传统方法对比：Forgetting/GraNd/EL2N 需要 $1.2C$-$2C$ 额外开销，Influence Functions/Data Shapley 需要 $10C$-$101C$+。CADS 在大预算下可低至 $\approx 2\gamma C$。[VERIFY: Table 6 数据]
- 且 CADS 的摊销特性意味着：一次选择可复用于多次训练。

### 3.3.3 CADS-E：样本级选择

**段落1 — 采样分布设计**
- 每个样本 $i$ 对应独立的 Bernoulli 参数 $s_i \in [0,1]$，表示入选概率。采样分布 $p(\mathbf{m}|\mathbf{s}) = \prod_i s_i^{m_i}(1-s_i)^{1-m_i}$。
- 对数梯度 $\partial \log p / \partial s_i = m_i/s_i - (1-m_i)/(1-s_i)$，计算简单。

**段落2 — 适用场景与特点**
- 优势：每个样本有独立的选择概率，可实现细粒度选择（如识别出特定的噪声样本）。
- 局限：参数量为 $N$（训练集大小），对大规模数据集存在优化效率问题。
- 适用于中小规模数据集或需要样本级精细控制的场景。

### 3.3.4 CADS-S：来源级选择

**段落1 — 采样分布设计**
- 将训练数据按来源分组，每个来源 $j$ 的采样比例 $r_j$ 服从截断高斯分布 $r_j \sim \mathcal{N}(s_j, \sigma_t^2)$，截断到 $[0,1]$。
- 方差退火：$\sigma_t = 0.99^t \sigma_0$，随训练逐步减小方差，使采样比例逐渐稳定。

**段落2 — 适用场景与关键行为**
- 适用于多源异构数据（如不同质量的数据集、不同域的数据混合），参数量仅为源数 $n \ll N$。
- 关键观察（CADS 的核心发现之一）：低预算时 CADS-S 几乎只选干净源（高噪声源权重趋近 0），高预算时逐步纳入低噪声源。极高噪声源（>60%）在所有预算下均被排除。[VERIFY: 具体阈值]
- 这一行为直接体现了数据选择与计算预算的交互：预算决定了模型能"消化"多大难度的数据。

## 3.5 讨论：Loss Predictor 的角色与局限性

**段落1 — CADS 对损失预测准确性的敏感性**
- CADS 的惩罚松弛依赖一维损失近似 $l(|\mathbf{m}|)$ 的准确性。如果该近似偏差过大，惩罚项会引导模型进入错误的训练状态。
- 分析何时对数线性近似可能失效：数据分布极端不均匀、预算范围跨越多个数量级、数据来源间质量差异极大时，简单的对数线性关��可能不够���确。
- 可能的改进方向：分段拟合、使用更灵活的函数族（如 scaling law 的四参数形式）。

**段落2 — 与 LLM 数据混合领域的联系**
- LLM 预训练中的核心问题之一：如何确定不同数据来源的混合比例（data mixture）？该问题与 CADS-S 的来源级选择在形式上高度一致。
- LLM 领域的 loss predictor / scaling law 方法（如 Chinchilla scaling laws、DoReMi）：通过预测不同配比下的损失来指导数据混合，与 CADS 中的一维损失近似 $l(|\mathbf{m}|)$ 扮演类似角色。
- 关键异同：CADS 显式引入计算预算作为变量，而大多数 LLM data mixture 工作假设固定预算（固定 token 数）。CADS 的预算感知思路可能对 LLM 领域有启发。

## 3.6 章节小结

**段落1 — 本章贡献总结**
- CADS 首次将计算预算纳入数据选择的优化框架，揭示了一个被广泛忽视的交互：最优数据子集是计算预算的函数。
- 在方法层面，通过策略梯度 + 惩罚松弛 + 一维损失近似，将昂贵的双层优化转化为高效的单层求解。
- 在实证层面，跨视觉和语言任务验证了预算感知数据选择的一致优势。

**段落2 — 向第四章过渡**
- CADS 优化数据选择时假设模型结构固定——模型作为"黑盒"接收所选数据并返回验证损失。但模型本身也存在冗余参数，而模型容量（参数量）与数据量之间存在已知的 scaling 关系。
- 自然的追问：当模型容量也成为可优化变量时（如通过剪枝压缩模型），数据选择与参数冗余之间是否存在类似的交互？第四章将回答这一问题。
